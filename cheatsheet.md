# atle vim cheat sheet
## File operations

| desc                 | hotkey   | command  |
| -------------------- | -------- | -------- |
| Nerdtree Toggle      | \<C-n\>  |          |

| Ag silver search     |          | :Ag      |

## Window
| desc                   | hotkey           | command  |
| ---------------------- | ---------------- | -------- |
| Split vertically       | C-w s | :split   |
| Split horizontally     | C-w v | :vsplit  |
| move focus right       | C-w l |          |
| move focus left        | C-w h |          |
| move focus up          | C-w k |          |
| move focus down        | C-w j |          |
| increase width         | C-w > |          |
| decrease width         | C-w < |          |
| increase height        | C-w + |          |
| decrease height        | C-w - |          |
| equal height and width | C-w = |          |
